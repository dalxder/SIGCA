    ArcGIS puede administrar datos de diferentes bases de datos como SQL server. Admin SQL server

    Requisitos de la base de datos de Microsoft SQL Server para ArcGIS 10.3.x info

    Versiones de base de datos compatibles

    Ediciones Standard/Enterprise

    • Microsoft SQL Server 2014 (64 bits)
    • Microsoft SQL Server 2012 SP1 (64 bits)
    • Microsoft SQL Server 2008 R2 SP2 (64 bits)
    • Microsoft SQL Server 2008 SP3 (64 bits)

    Ediciones Express

    • Microsoft SQL Server 2014 (32 bits* y 64 bits)
    • Microsoft SQL Server 2012 SP1 (32 bits* y 64 bits)
    • Microsoft SQL Server 2008 R2 SP2 (32 bits* y 64 bits)
    • Microsoft SQL Server 2008 SP3 (32 bits* y 64 bits)
    *Microsoft SQL Server Express (32 bits) solo se admite para geodatabases de escritorio. info
