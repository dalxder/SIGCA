Cantidad de agua

El sistema de información hidrológica SIH, es una de las herramientas informaticas que posee el área de hidrológia especializada, que permite realizar la gestión de la información hidrologica de la empresa, esencial para la toma de decisiones en el diseño y operación de los sistemas de abasteciminetos y drenaje de la EAB.

El objetivo inicial fue el de "construir una base de datos hidrológica, dotada de mecanismos modernos para el procesamiento, análisis y evaluación de la información hidrológica, como materia prima de los objetivos corporativos.

El SIH es principalmente una base de datos, con herramientas que permiten realizar diversas operaciones sobre estas.

Las herramientas "programas aplicativos", consumen información contenida en la base de datos, permitiendo realizar operaciones de tipo CRUD (crear, leer, actualizar y eliminar datos) sobre esta.

Historia del SIH

El SIH fue desarrollado originalmente en 1996, en el 2003 se llevo a cabo una actualización con el objetivo de habilitar su operación en la web (Navegador web).

La segunda version del sistema se encuentra en operación actualmente por parte del personal de hidrológia especializada, pero este sistema requiere de algunas actualizaciones, el sistema no recibe mantenimiento regular desde el año 2007. La versión funcional vigente esta desarrollada en JAVA J2EE.

En el desarrollo del SIH se realizo de acuerdo a los requerimientos que deberia tener el sistema (Anexo: Requerimientos funcionales)

EL SIH original, así como las actualizaciones tecnológicas, fueron diseñadas y desarrolladas por el ingeniero JUAN MANUEL RAMÍREZ CORREDOR.
