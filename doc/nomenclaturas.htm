Nomenclaturas

Ajax: Acrónimo de Asynchronous JavaScript And XML

API: Acrónimo de las palabras en ingles "Aplication protocol interface".

POJO: Acrónimo de Plain Old Java Object

CRUD: Acrónimo de las palabras en ingles "create", "read", "update" y "delete".

REST:Acrónimo de las palabras en ingles "Representational State Transfer".
